# Customer-Churn-Prediction-System
AI-powered customer retention analytics platform with real-time churn prediction and interactive dashboard visualization.
📌 Customer Churn Prediction System
🚀 Overview

This project is a production-style Machine Learning system that predicts customer churn risk and visualizes customer analytics through a real-time dashboard.

The system combines:

Machine Learning
Explainable AI
FastAPI backend
Next.js frontend
Real-time analytics
🧠 Problem Statement

Businesses lose revenue when customers stop using their services.

This project predicts:

Which customers are likely to churn in the next billing cycle.

The system helps organizations:

identify high-risk users
prioritize retention campaigns
improve customer lifetime value

Category	Technology
